# Обзор ограничений PROMO-линейки

Тарифы PROMO предназначены как бюджетное решение для некритичных задач, например разработки ПО, обучения по работе с серверами и личных проектов.

Чтобы сохранить низкую стоимость, на данную линейку распространяется ряд ограничений.

---

## Ограничения SLA и поддержки
- Отсутствует SLA технической поддержки и гарантия аптайма.
- В тариф не входит техническая поддержка; при возможности может быть предоставлена только базовая консультация. В случае проблем со стороны компании, например недоступности сервера в связи с нашей стороны, обращения в поддержку разрешены, и компания всегда предпримет всё возможное для скорейшего решения вопроса.
- Обращения в поддержку связанные с техническими вопросами не игнорируются, однако время ответа будет значительно дольше, чем на стандартных тарифах.
- Сложные задачи (установка ПО, настройка, диагностика, отладка) **не выполняются** отделом поддержки.
- Партнёрские начисления с реферальной программы и скидки не применяются к тарифам данной линейки.

## Ограничения операционных систем и установки
- Поддержка ОС Windows отсутствует.
- Монтирование кастомных ISO недоступно.
- Разрешено использовать только предоставляемые нами шаблоны ОС.

## Ограничения по производительности
- Используется общий NVMe-накопитель, с различными лимитами IOPS, в зависимости от выбранного тарифа:  
 DE-PROMO-1 — **8k IOPS** на VPS  
 DE-PROMO-2 — **16k IOPS** на VPS  
 DE-PROMO-3 — **24k IOPS** на VPS  
 DE-PROMO-4 — **32k IOPS** на VPS  
- Используется fair-share модель деления CPU, с гарантированной долей **до 10%** от vCPU на узле.
- Длительная высокая (100%) загрузка сети может привести к шейпингу согласно **[политике честного использования](https://senko.digital/acceptable-use-policy)**.

## Сетевые ограничения
- Пропускная способность ограничена характеристиками тарифа.
- Исходящие почтовые порты (SMTP 25/465/587) **не могут быть открыты**.
- Постоянная загрузка канала **запрещена**.

## Рекомендуемые сценарии использования
Эти тарифы идеально подходят для:
- разработки и тестирования ПО
- стейджинговых окружений
- временных или экспериментальных проектов
- личного VPN
- персонального сайта или простого приложения
- обучения, практики и освоения Linux

Данная тарифная линейка **не предназначена** для сервисов, требующих высокий аптайм и гарантированные ресурсы.
